Schrijf een functie telOp (a, b) die twee getallen optelt en de som retourneert. Verwachte Output: 8

Schrijf de volgende functie in een JavaScript bestand.

function telOp(a, b) {
	return a + b;
}
console.log(telOp(5, 3));

Uitleg

Hier definiÃ«ren we een functie genaamd telOp. De functie neemt twee parameters: a en b. Dit zijn de getallen die we willen optellen.

Binnen de functie gebruiken we het return-statement om de som van a en b terug te geven. De expressie a + b voegt de waarden van a en b samen. Dit betekent dat als je de functie aanroept met twee getallen, het resultaat de som van die getallen is.

console.log wordt gebruikt om het resultaat van de functie naar de console te schrijven, zodat je het kunt zien.

Tip

Open de console in je browser om de uitkomst te zien. Druk op F12 of Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
