Functie voor even en oneven getal

Terug naar Home
Schrijf een functie isEven(getal) die controleert of een getal even is en true retourneert als dat zo is, anders false.

Schrijf de volgende functie in een JavaScript bestand.

function isEven(getal) {
	returngetal % 2 === 0;
}
console.log(isEven(6));
console.log(isEven(7));

Uitleg

De functie isEven is een parameter die getal accepteert. Dit is het getal dat we willen controleren.

Het return sleutelwoord gebruikt de modulusoperator (%) om de rest van de deling van getal door 2 te berekenen. Als het resultaat 0 is, betekent dit dat het getal even is. De expressie getal % 2 === 0 retourneert true als het getal even is en false als dat niet zo is.

Met console.log testen we de functie met twee verschillende getallen.

  • De eerste test (isEven(6)) zou true moeten retourneren omdat 6 een even getal is.
  • De tweede test (isEven(7)) zou false moeten retourneren omdat 7 een oneven getal is.

Tip

Open de console in je browser om de uitkomst te zien. Druk op F12 of Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
