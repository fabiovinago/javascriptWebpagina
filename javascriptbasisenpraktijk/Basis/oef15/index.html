Schrijf een functie keerOm(tekst) die een string omkeert en de omgekeerde string retourneert. Verwachte Output: "tpircSavaJ"

Schrijf de volgende functie in een JavaScript bestand.

function keerOm(tekst) {
	return tekst.split('').reverse().join('');
}
console.log(keerOm("JavaScript"));

Uitleg

De functie keerOm is een parameter die een parameter tekst accepteert. Dit is de string die we willen omkeren.

Om de string om te keren doen we het volgende

  • tekst.split(''): dit splitst de string in een array van afzonderlijke karakters. Bijvoorbeeld, "JavaScript" wordt ["J", "a", "v", "a", "S", "c", "r", "i", "p", "t"].
  • .reverse(): dit keert de volgorde van de elementen in de array om. De array wordt nu ["t", "p", "i", "r", "c", "S", "a", "v", "a", "J"].
  • .join(''): dit voegt de elementen van de omgekeerde array weer samen tot een enkele string zonder tussenruimtes. De resulterende string is "tpircSavaJ".

Met console.log Hier testen we de functie door de string "JavaScript" aan te roepen. We verwachten dat de output "tpircSavaJ" is, wat de omgekeerde versie van de oorspronkelijke string is.

Tip

Open de console in je browser om de uitkomst te zien. Druk op F12 of Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
