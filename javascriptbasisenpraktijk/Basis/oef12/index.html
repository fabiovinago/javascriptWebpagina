Schrijf een functie begroet(naam) die een naam als parameter neemt en “Hallo, (naam)!” retourneert. Verwachte Output: "Hallo, Jan!"

Schrijf de volgende functie in een JavaScript bestand.

function begroet(naam) {
	return "Hallo " + naam + "!";
}
console.log(begroet("Jan"));

Uitleg

(begroet("Jan")) is het sleutelwoord voor de function dat aangeeft dat we een functie gaan definiëren en begroet is de naam van de functie. We kunnen deze naam gebruiken om de functie aan te roepen.

(naam) is de parameter van de functie. Het betekent dat wanneer we de functie aanroepen, we een waarde (in dit geval een naam) moeten doorgeven. Deze waarde wordt binnen de functie beschikbaar als de variabele naam.

console.log wordt gebruikt om het resultaat van de functie naar de console te schrijven, zodat je het kunt zien.

Tip

Open de console in je browser om de uitkomst te zien. Druk op F12 of Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
