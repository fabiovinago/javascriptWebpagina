Functie klinkers vervangen

Terug naar Home
Schrijf een functie vervangKlinkers(tekst) die alle klinkers in een string vervangt door het teken *. Verwachte Output: "H*ll* W*r*ld"

Schrijf de volgende functie in een JavaScript bestand.

function vervangKlinkers(tekst) {
	return tekst.replace(/[aeiouAEIOU]/g, '*';
}
console.log(vervangKlinkers("Hallo Wereld"));

Uitleg

De functie vervangKlinkers is een parameter die tekst accepteert. Dit is de string waarin we de klinkers willen vervangen.

Om de tekst te vervangen door * doen we het volgende

  • tekst.replace(...): De replace-methode wordt gebruikt om delen van de string te vervangen.
  • /[aeiouAEIOU]/g: Dit is een reguliere expressie die alle klinkers (zowel kleine als hoofdletters) zoekt.
    • a, e, i, o, u zijn de klinkers, en de hoofdletters A, E, I, O, U zijn toegevoegd om ook de hoofdletters te vervangen.
    • De g (global) vlag zorgt ervoor dat alle voorkomens van de klinkers in de string worden vervangen, niet alleen de eerste.
  • '*'-teken: Dit is het teken waarmee de klinkers worden vervangen.

Met console.log testen we de functie door de string "Hallo Wereld" aan te roepen. De verwachte output is "H*ll* W*r*ld", waarbij alle klinkers zijn vervangen door *.

Tip

Open de console in je browser om de uitkomst te zien. Druk op F12 of Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
