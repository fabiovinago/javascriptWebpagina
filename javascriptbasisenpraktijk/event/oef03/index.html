Voeg een click event toe aan een knop

Terug naar Home
Maak een HTML knop met de tekst "Klik hier!" en geeft deze knop een unieke ID (mijnKnop)

Schrijf de volgende HTML-code in de index.html file

<button id="mijnKnop">Klik hier!</button>

Je bekomt het volgende resultaat

Schrijf een click event in een afzonderlijke js.index Javascript file

const knop = document.getElementById('mijnKnop');
knop.addEventListener('click', function() {
console.log("Knop is geklikt");
});

Uitleg

Uitleg van de HTML-code

  • De button-tag maakt een knop met de tekst "Klik hier!" en geeft deze knop een unieke ID (mijnKnop) zodat we deze later kunnen selecteren met JavaScript.

Uitleg van de JavaScript-code

  • Met document.getElementById selecteren we de knop in de DOM (Document Object Model).
  • We voegen een "click" event listener toe aan de knop. Dit betekent dat wanneer de knop wordt geklikt, de functie wordt uitgevoerd die het bericht "Knop is geklikt!" in de console logt.
  • Wanneer je de pagina opent in een webbrowser en op de knop klikt, zie je de output "Knop is geklikt!" verschijnen in de console.
