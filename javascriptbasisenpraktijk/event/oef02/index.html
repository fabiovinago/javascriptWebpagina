Schrijf een functie vierkant(getal) die een getal kwadrateert en het resultaat retourneert. Verwachte Output: 16

Schrijf de volgende functie in een JavaScript bestand.

function vierkant(getal) {
	return getal * getal;
}
console.log(vierkant(4))

Uitleg

Hier definiÃ«ren we de functie vierkant, die een parameter getal accepteert. Dit is het getal dat we willen kwadrateren.

Het return sleutelwoord vermenigvuldigt het getal met zichzelf. Het resultaat van deze vermenigvuldiging (het kwadraat van het getal) wordt geretourneerd.

Met console.log roepen de functie vierkant aan met het argument 4. De verwachte output is 16, omdat 4 * 4 = 16. Deze waarde wordt naar de console gelogd.

Tip

Open de console in je browser om de uitkomst te zien. Druk op F12 of Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
